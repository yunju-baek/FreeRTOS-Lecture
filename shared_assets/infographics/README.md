# Shared FreeRTOS Lecture Infographics

Created: 2026-05-06

## Usage Policy

- Store reusable generated infographics here.
- Internal labels may be generated as pixel text inside the image.
- Avoid HTML overlays for labels inside complex images because alignment is fragile.
- Do not include an overall infographic title, slide title duplicate, figure number, footer, or bottom figure caption inside the image.
- Keep embedded labels short and large enough to remain readable after slide insertion.
