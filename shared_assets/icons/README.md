# Shared FreeRTOS Lecture Icons

Created: 2026-05-06

## Usage

- Reusable icons for English FreeRTOS HTML lecture decks.
- Keep icons in this shared folder when they can be reused across modules.
- Insert with `object-fit: contain`, `height: auto`, and a fixed layout slot so the original ratio is not distorted.
- Do not create fake icons by placing abbreviations or letters inside circles.
